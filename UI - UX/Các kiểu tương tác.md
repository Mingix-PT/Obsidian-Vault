# Dòng lệnh
- Ưu điểm:
	- Thích hợp với nhiệm vụ tính lặp
	- Thích hợp với ND có kinh nghiệm do tính ngắn gọn, nhanh và dễ hiểu của câu lệnh
- Nhược điểm:
	- Cần đào tạo
	- Sai sót cao
	- Khó xử lý lỗi

# Menu
